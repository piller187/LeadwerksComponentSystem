Background

The project started when I (Roland) read a blog post "Top 10 celebrities who use component architecture!" written by Rick at the Leadwerks Community.

I found Rick's ideas quite fascinating and started a private discussion with him about he's ideas. One thing lead to another and slowly we began to build a system for implementing this with Leadwerks in mind.

So what is this system named Leadwerks Component System (or shorter LCS) then.

LCS is a way to decrease the complexity of programming and increase the amount of reusable code. It's a LUA only project and free to use for anyone who like it

LCS consist of 3 main class types

  • Component
  • Message
  • Game Object
